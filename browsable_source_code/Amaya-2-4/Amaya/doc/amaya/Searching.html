Searching and replacing text

You can search and replace text with the Search dialog box. Call this box by choosing Find from the Edit menu, click on the corresponding button , or use the keyboard shortcut (control-shift-% in Unix, control-f in Windows)

This form allows you to search or replace text using different criteria:

  • The character string to be searched for must be entered in the Search for input area.
  • If the UPPER=lower button is selected, a case-insensitive search is performed; otherwise, the string is searched for as it appears within the "Search for" input area.
  • Strings can also be replaced using the same operation. The replacement text is entered in the Replace by input area and a replacement mode is selected from the Replace menu. The replacement modes are as follows:
    • No replace: no replacement occurs even if the ``Replace by'' input area contains a character string. It is just a simple search.
    • Replace on request: when the string is found, it is selected. The user can then select one of the following buttons:
      • Confirm to replace and carry out the search again.
      • Do not replace to search for the next instance without replacing the string which has been found.
    • Automatic replace: all instances of the string searched for are automatically replaced within the portion of document defined in the "Search where" menu.
  • The Search where menu allows you to specify the part of document in which the search occurs:
    • Before selection: the search starts at the beginning of the selected part and moves backwards to the beginning of the document.
    • Within selection: the search only occurs within the selected part, from its beginning to its end.
    • After selection: the search starts at the end of the selected part and moves forwards to the end of the document.
    • In the whole document: the search is carried out in the whole document from the beginning to the end regardless of which part is selected.

To initiate the search operation, click on the Confirm button at the bottom of the form. If the string searched for is found, it is selected and the document is positioned so as to make this string visible; you can then search for the next instance of this string by selecting the Confirm button again. If the string is not found, the "Not found" message appears in the bottom right-hand corner of the form.

Search or replace can be abandoned at any moment by clicking on the Done button.

New: Replacement operations can now be undone with the Undo function (in the Edit menu, or keyboard shortcut control-z).
